# Karthik B Portfolio

A modern and responsive developer portfolio built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Features

- Responsive Design
- Smooth Animations
- Resume Download
- Project Showcase
- Contact Section
- Dark Crimson Noir Theme
